# 📋 변경 로그 (Changelog)

이 파일은 GST 공장 불량 예측 시스템의 모든 주요 변경사항을 추적합니다.

## [v2.3.0] - 2026-01-06

### ✨ 추가 (Added)
- **연도 선택 드롭다운**: 대시보드 헤더에 2025년/2026년 전환 기능 추가
- **다년도 데이터 분리**: 2025년과 2026년 데이터를 별도 페이지로 분리 운영
- **한국 시간대(KST) 적용**: 업데이트 타임스탬프를 UTC에서 KST로 변경
- **주차별 분석 탭**: 주간 트렌드 분석 기능 추가

### 🔧 개선 (Changed)
- **Plotly 드롭다운 스크롤바**: 가시성 향상을 위한 스크롤바 크기 및 스타일 개선
- **호버 툴팁 개선**: 4주차 트렌드 차트의 hovermode를 "closest"로 변경
- **Teams 파일 경로 수정**: `General/99.개인폴더/박승록/0. 불량&TIME`으로 변경
- **2026년 데이터 소스**: `▶2026年 검사 통합 Sheet.xlsm` 파일 연동
- **ML 분석 데이터 분리**: 월간 ML 분석은 2025년 데이터 사용 (데이터 충분)
- **환경변수 오버라이드**: `TEAMS_EXCEL_FILE_NAME`으로 파일명 동적 변경 가능

### 🏗️ 구조 (Structure)
- **URL 구조**:
  - 2026년: `public/internal.html` (기본)
  - 2025년: `public/2025/internal.html`

### 🧪 테스트 (Testing)
- **로컬 테스트 완료**: Teams API 연동 및 2026년 데이터 정상 로드 확인
- **GitHub 배포 완료**: https://isolhsolfafa.github.io/gst-factory/public/internal.html

## [v2.2.1] - 2025-09-16

### ✨ 추가 (Added)
- **불량위치 정보 hover**: "부품별 검사공정 비교(TOP5)" 차트에 불량위치 상세 정보 추가
- **향상된 툴팁 시스템**: 월별 데이터 포인트에 주요 불량위치 3개까지 표시
- **동적 hover 컨텐츠**: 가압검사/제조품질별로 개별 불량위치 데이터 제공

- **데이터 접근성**: 별도 drill-down 없이 즉시 상세 정보 확인
- **정보 밀도 최적화**: 조치내용 제외, 불량위치만 표시하여 가독성 향상

### 🧪 테스트 (Testing)
- **실제 데이터 검증**: Teams API 연동 (가압: 391건, 제조품질: 208건)
- **호버 기능 테스트**: 모든 TOP5 부품에 대한 불량위치 정보 정상 표시
- **GitHub 배포 완료**: https://isolhsolfafa.github.io/gst-factory/public/internal.html

## [v2.2.0] - 2025-09-15

### ✨ 추가 (Added)
- **부품별 검사공정 비교 차트**: TOP5 부품의 가압검사/제조품질 분리 분석 기능
